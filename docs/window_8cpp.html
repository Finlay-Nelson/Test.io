Detailed Description

Utilises the Window class to generate the game GUI, as well as process and respond to incoming data.

This demo is to test the self-written ads1115 lib to sample continuously. No such lib for the ads1115 in c++ could be found. In this test demo the ADC sampling is controlled by timer instead of by interrupt. This is to separate the lib and expose any bugs.
