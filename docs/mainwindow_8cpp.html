Detailed Description

Realisation of the EMG-controlled Pong GUI.

Author
Zonghan Gan
Finlay Nelson
Henry Cowan
Date
11 April 2020

Uses the mainwindow class defined in mainwidnow.cpp and the Qt toolkit to generate, update and control the Pong GUI according to incoming EMG signals. The MainWindow() function is used to instantiate the game window, and generate a QGraphicsScene() object in which to place the GUI items. The server socket and host ip address are bound to allow UDP communication.
