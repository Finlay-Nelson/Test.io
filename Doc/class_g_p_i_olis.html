Detailed Description

This class is responsible for the data communications and threading involved with conversion and processing of input data to the ADS1115.

The GPIOlisten class monitors the ALERT/RDY pin and triggers an interrupt each time an AD conversion is ready. The data is then processed before sleeping until the next interrupt. Important member functions include:

  • GPIOlis()
  • interrupt()
  • run()
  • quit()

◆ interrupt()

void GPIOlis::interrupt ( void  )

Emits the readyread() signal with a measured voltage in response to an interrupt.

Uses the readsig() function of the ads1115 class to measure an input voltage before emitting the readyread() signal containing the voltage result. A counter keeps track of the number of conversions performed (uses of the interrupt() function), each time alerting the onset of conversion through the qDebug() slot.
