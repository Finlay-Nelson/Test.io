Detailed Description

ADS1115 Bit[7-5].

These bits control the data rate setting.

  • 8 sps
  • 16 sps
  • 32 sps
  • 64 sps
  • 128 sps
  • 250 sps
  • 475 sps
  • 860 sps

The DEFAULT data rate is 250 samples per second.
